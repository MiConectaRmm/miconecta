# Branding assets

Coloque os arquivos nestes caminhos:

## Windows / agente
- `agent/MIConectaAgent.Tray/icon.ico`
  - `.ico` multi-resolução com: 16, 24, 32, 48, 64, 128, 256

## Frontend / portal
- `frontend/public/favicon.ico`
  - `.ico` com: 16, 32, 48
- `frontend/public/branding/logo-horizontal.svg`
  - alternativa: `logo-horizontal.png` em 1200x300
- `frontend/public/branding/logo-mark.svg`
  - alternativa: `logo-mark.png` em 512x512
- `frontend/public/branding/logo-dark.svg`
  - alternativa: `logo-dark.png` em 1200x300
- `frontend/public/branding/logo-light.svg`
  - alternativa: `logo-light.png` em 1200x300
- `frontend/public/branding/icon-192.png`
  - 192x192
- `frontend/public/branding/icon-512.png`
  - 512x512

## Instalador / materiais
